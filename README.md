# 🧩 Sudoku Game

An interactive and fun Sudoku game built using **HTML, CSS, and JavaScript**. The game features difficulty levels, dark mode, a timer with pause functionality, and an intuitive user interface.

---

## 🎮 Features

- ✅ **3 Difficulty Levels**: Basic, Medium, and Extreme
- ⏱ **Timer**: Starts with each game and includes a Pause/Resume option
- 🧠 **Smart Highlighting**: Highlights same numbers for better visualization
- 🏠 **Home & Restart Buttons**: Return to the home screen or restart a puzzle
- ❌ **Input Validation**: Accepts only numbers between 1–9

---

## 🖥️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
